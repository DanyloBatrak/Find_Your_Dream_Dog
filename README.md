# Web Development Project 4 - Find_Your_Dream_Dog

Submitted by: Danylo Batrak

This web app: This web app allow you to generate image of dog with attached attributes: breed, life_span, temperament and origin. Inside the web you can click on attributes to ban them if you not like such attributes to generate image of dog with different attributes. After you click "Generate" button once again, this generated image that been created before with its created attributes would appear in other window called "What you found so far" with image of previous dog and with detailed description.   

Time spent: 11 hours spent in total
